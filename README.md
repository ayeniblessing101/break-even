# gRPC Calculate Break Even

Follow these setup to run the to get the App up and running:

1.  Get the code:

    ```console
    $ git clone https://github.com/ayeniblessing101/break-even.git
    $ go get ./...
    ```

2.  Run the server:

    ```console
    $ cd server
    $ go run server.go
    ```

3.  Run the client:

    ```console
    $ cd client
    $ go run client.go
    ```
